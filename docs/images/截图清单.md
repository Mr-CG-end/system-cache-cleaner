# 操作手册截图清单

请使用 PNG 格式并保持窗口完整可见，不要裁掉标题栏、底部按钮或对话框边缘。

如果此前运行过旧版演示脚本，请先在 Demo 模式中扫描、全选并清理旧演示数据，然后重新运行当前 `Prepare-DemoData.ps1`，以免旧文件进入截图统计。

| 文件名 | 启动模式 | 截图时机 |
| --- | --- | --- |
| `01_main_window_idle.png` | 普通模式 | 启动后尚未扫描，显示完整主窗口 |
| `02_demo_mode_banner.png` | 演示模式 | 带 `--demo` 启动后，突出顶部黄色演示横幅 |
| `03_scan_completed_list.png` | 演示模式 | 准备演示数据并完成扫描，显示三个类别和底部汇总 |
| `04_cleanup_confirm_dialog.png` | 演示模式 | 在图 3 后勾选 Windows 临时文件，三个类别全选，再点击“立即清理” |
| `05_cleanup_report_dialog.png` | 演示模式 | 全选清理完成后，显示 8 个成功项、16.68 MB 和开始/结束时间 |
| `06_rescan_completed.png` | 演示模式 | 在报告中点击“重新扫描”后，三个类别均显示 0 个文件和 0 B |

截图文件请直接放在本目录。六张图片齐全后，再生成最终 Word 操作手册。
